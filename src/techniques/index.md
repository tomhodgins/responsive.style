# Techniques

This collection of styling techniques goes beyond the limits of what CSS can do. This list attempts to expose the most useful ways JavaScript and CSS can be used together, and to explain the required parts to make each technique work.

Most of the code samples used in these pages will require the use of one of the [Plugins](../plugins/) from the Plugins section, but all of these techniques can also be built from scratch using JavaScript.
